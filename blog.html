1.var: The var keyword declares a function-scoped or global variable. Function-scoped means that the variable is only available within the function it was declared in. Global variables are available throughout your entire code .

2.let: The let keyword declares a block-scoped local variable.Block-scoped means that the variable is only available within the block it was declared in

3.The const keyword declares a block-scoped.a variable that can't be reassigned Constants are also called “immutable variables”, but that's a bit of a misnomer since they are actually variables - just ones that can't be reassigned.

1.Unlike regular functions, arrow functions do not have their own this. The value of this inside an arrow function remains the same throughout the lifecycle of the function and is always bound to the value of this in the closest non-arrow parent function.

2.Regular functions created using function declarations or expressions are constructible and callable. Since regular functions are constructible, they can be called using the new keyword. However, the arrow functions are only callable and not constructible, i.e arrow functions can never be used as constructor functions. Hence, they can never be invoked with the new keyword.

3.Filter checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria.
